# AK Token Ecosystem

Official static website for the **AK Token Ecosystem**, a community utility project built on **Polygon Network**.

## Project Overview

AK Token Ecosystem presents a public transparency dashboard for:

- **Polygon Network** token transparency
- **NFT Collections** connected to the AK Token ecosystem
- **Community Utility Project** positioning
- **Public Transparency Dashboard** for holders, transfers, contract, treasury, and ecosystem links

## NFT Collections

The website includes official OpenSea links for:

- Jawa Cosmic Legends
- Zurp #001
- Zurp #002
- Zurp #003 The Foes of Prince Zoq
- Javanese Legends Collection
- Putri Bali
