How To Buy / Swap AK Token

  1. Install MetaMask Wallet or Trust Wallet.

  3. Deposit POL token for swap and gas fee.
  4. Open Uniswap on Polygon.
  5. Paste the AK Token contract address.
  6. Review token contract carefully before swapping.
  7. Swap POL to AK only if you understand the risk.
